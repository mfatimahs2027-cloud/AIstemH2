Our Mission

This project was created to share fun, interesting, and educational information about animals in a simple and engaging way. Each slide highlights a different animal group with facts, traits, and unique characteristics.

How It Was Built

The website uses clean HTML structure and organized sections to make the content easy to read. It is designed to be expanded with more animals, images, and educational features in the future.
